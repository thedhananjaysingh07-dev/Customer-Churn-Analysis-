# 📊 Customer Churn Analysis

## 🔹 Project Overview
This project analyzes customer churn behavior in a telecom company to identify key factors affecting customer retention and revenue loss.

## 🔹 Tools Used
- Excel (Data Cleaning)
- MySQL (Data Analysis)
- Power BI (Data Visualization)

## 🔹 Objectives
- Identify factors leading to customer churn
- Analyze customer behavior patterns
- Measure revenue impact of churn
- Provide actionable insights for retention

## 🔹 Key Metrics
- Total Customers
- Churned Customers
- Churn Rate (%)
- Total Revenue
- Revenue Lost

## 🔹 Key Insights
- Month-to-month contracts show the highest churn rate  
- Customers with low tenure are more likely to churn  
- Higher monthly charges increase churn probability  
- Significant revenue is lost due to churned customers  
- Long-term contracts improve customer retention  

## 🔹 Dashboard Features
- Interactive slicers (Contract, Gender, Internet Service, Tenure)
- KPI cards for business metrics
- Visual analysis of churn drivers
- Revenue impact visualization
